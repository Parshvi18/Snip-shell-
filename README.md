# SnipShell

SnipShell is a secure code snippet management platform.

## Features
- User authentication with JWT
- Create code snippets
- View personal snippets
- Delete snippets
- Secure API routes

## Tech Stack
Node.js
Express.js
MongoDB
JWT Authentication

## API Endpoints
POST /api/auth/signup
POST /api/auth/login
POST /api/snippet
GET /api/snippets
DELETE /api/snippet/:id
